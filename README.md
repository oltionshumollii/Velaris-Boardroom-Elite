# 🚀 Velaris Boardroom Elite

A modern, ultra-professional corporate landing page built with pure HTML, CSS, and JavaScript.

Designed with luxury UI principles, smooth animations, dark/light mode, and interactive elements — optimized for performance and responsiveness.

---

## 🌟 Live Features

- 🎨 Smart Dark / Light Mode (auto-save preference)
- 📱 Fully Responsive (Mobile + Tablet + Desktop)
- 📊 Animated Counters
- ✨ Scroll Reveal Animations
- 📜 Scroll Progress Bar
- 📂 Mobile Hamburger Menu (smooth animation)
- 🧠 3D Background Canvas Effect
- 📩 Contact Form with Real Validation
- 🔔 Toast Notifications
- 🏢 Professional Corporate UI Design

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom UI System)
- Vanilla JavaScript (No frameworks)
- Google Fonts (Inter)
- SVG Icons
